# Sistema de Listagem de Clientes

Sistema simples em JavaScript para gerenciar clientes utilizando funções construtoras, com suporte a criação de objetos Cliente, TelefoneCelular e Endereco, incluindo métodos de acesso (getters/setters) e ordenação alfabética.

## Como executar

```bash
node index.js
```

O script irá exibir informações formatadas dos clientes e demonstrar a ordenação alfabética por nome.
